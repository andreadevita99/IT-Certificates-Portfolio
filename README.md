# 📜 IT Certificates Portfolio – Andrea De Vita 🎓

An interactive gallery that collects and displays all of my IT certifications, featuring previews, full-screen zoom, and dark mode support.

## ✨ Key Features

- **Responsive gallery** – a grid layout that adapts to desktops, tablets, and smartphones.
- **Center zoom** – when you click on a card, the certificate image zooms in to the center of the page with a smooth animation.
- **Auto-close** – after 5 seconds, the enlarged view closes automatically; you can also close it by clicking on the background.
- **Dark / Light mode** – iOS-style toggle (sun/moon) to switch between light and dark modes; the preference is saved in the browser.
- **Custom favicon** – integrated via jsDelivr from the GitHub repository.
- **Optimized performance** – images are loaded lazily to avoid slowing down the page.

## 🛠️ Technologies used

- **HTML5** – semantic structure
- **CSS3** – CSS variables for themes, animations, Flex/Grid
- **JavaScript (Vanilla)** – gallery management, modals, dark mode, and timers
- **Google Fonts** – *Inter* font for modern typography
- **jsDelivr** – CDN for the favicon

## 🌗 Dark mode
The toggle in the top-right corner lets you switch between light and dark mode.
Your selection is saved in localStorage, so it remains active even after you reload the page.
